== What You Need

:java_version: 15

----
./mvnw spring-boot:run
----

The actuator exposes the following:

* http://localhost:8080/file
* http://localhost:8080/file/{id}
* http://localhost:8080/file/{id}
* http://localhost:8080/file/{id}/download
* http://localhost:8080/files
